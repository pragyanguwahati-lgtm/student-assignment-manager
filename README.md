# Student Assignment Manager (SAM)

## Project title
Student Assignment Manager (SAM)

## Overview
SAM is a lightweight Python-based CLI application for managing students, assignments, submissions, and basic analytics. It uses SQLite as local storage and is intended as a course project for introductory Python programming.

## Features
- Add / list / delete students
- Create / list / update / delete assignments
- Record submissions and record grades/comments
- Compute average grade per assignment and submission rate
- Export assignment report to CSV

## Technologies / Tools used
- Python 3.10+
- SQLite (via `sqlite3`)
- pytest (for unit testing)
- VS Code (recommended editor)
- Git & GitHub for version control
